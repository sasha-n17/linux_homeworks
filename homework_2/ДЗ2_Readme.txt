1. Для быстрого перехода в домашний каталог можно пользоваться следующими параметрами cd:
a) cd 
b) cd ~
c) cd /home/aleksandr/

Для того, чтобы подняться на уровень выше используется команда: cd ../
На два уровня выше: cd ../../
и так далее.
Все соответствующие скриншоты будут представлены в папке homework_2_screens.
